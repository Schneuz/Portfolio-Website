# Mission
Implementiere die Website im Stil Quiet Technical Editorial.

# Zielwirkung
Die Website muss:
- professionell
- modern
- technisch glaubwürdig
- ruhig
- nicht trocken
- nicht verspielt

wirken.

# Nicht tun
- keine Stil-Mischung aus mehreren Referenzseiten
- keine section-spezifischen Farbwelten
- keine spielerischen Sonderformen
- keine visuelle Überinszenierung
- keine überladene Motion
- keine Designentscheidungen, die nur auf einem Screenshot gut aussehen

# Immer tun
- halte Farben, Typografie, Kartenlogik, Spacing und Linienführung konsistent
- nutze ein ruhiges helles Grundsystem
- nutze genau eine dominante Akzentfarbe
- nutze große, klare Typografie
- setze Struktur vor Dekoration
- vereinfache komplexe Sektionen auf Mobile konsequent

# Stilregeln
- off-white Hintergrund
- dunkler, fast schwarzer Text
- feine Borders
- 24px Card-Radius
- subtile Shadows
- großzügiges Spacing
- editoriale, technische Ruhe

# Komponentenlogik
- gleiche Card-Sprache in allen Card-Sektionen
- gleiche Button-Sprache in allen CTA-Elementen
- gleiche Heading-Logik in allen Sektionen
- Linien nur als strukturelles Mittel, nicht als Dekoration

# Responsive Regel
Wenn die Referenz auf Mobile nicht sinnvoll ist:
- baue eine responsive Interpretation
- nicht die Desktop-Version krampfhaft verkleinern

# Entscheidungsfrage
Bevor du visuell etwas hinzufügst, prüfe:
1. Macht es die Seite klarer?
2. Passt es zur ruhigen technischen Gesamtästhetik?
3. Ist es systemkonform?
4. Funktioniert es auf Mobile?
